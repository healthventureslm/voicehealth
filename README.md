# VoiceHealth

Sistema de gestão de qualidade clínica e metas internacionais de segurança do paciente.

## Stack
- React + Vite + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (Postgres, Auth, Edge Functions, Storage)

## Desenvolvimento

```bash
npm install
npm run dev
```

Configure o `.env` com as variáveis do seu projeto Supabase (veja `.env.example`).
